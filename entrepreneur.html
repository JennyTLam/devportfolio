Meet the Team

Memento began as an incubating team at The Garage while its founders where undergraduates at Northwestern University. The project started as a Practicum for Intelligent Information Systems and has iteratively grown to explore cloud based face recognition systems, natural language processing, and Google Glass development. Jenny Lam, Andrew Acomb, Daniel Chang, Diana Smith, and Elise Lee make up the current Memento development team. For inquiries please email themomentoproject@gmail.com.

About Memento

Memento

Memento is a team dedicated to creating more equitable facial-recognition experiences for visually-impaired or memory-impaired users. Our core product is a facial-recognition headpiece, built on Google Glass, that helps users match faces of people they’ve met to their names. Facial recognition through the Kairos API, a backend system hosted on an AWS EC2 instance, bone conduction technology, and a React companion app all work together to create a helpful and robust experience for users.

How it Works

The Memento system integrates Google Glass, Amazon Web Services, a companion phone app, and Kairos a cloud-based face recognition platform. When a user meets someone, Memento can be prompted to take an image capture and send the picture to the server for processing. The server then makes calls to the Kairos API and returns to the glass a text string representing the name of the person the user met. Google Glass then outputs the name string as audio using Google Text-to-Speech and the Glass's bone conduction technology. Names can be updated via the companion app and other speech-enabled commands.

VentureCat Pitch Competition

Competition Team

The Memento team competed in the 2021 VentureCat pitch competition, Northwestern University's annual student startup competition. Part of the Social Impact and Nonprofit track, Memento aims to provide accessible technology to help users with visual or memory related disorders better recognize others and their surroundings.

The Garage Northwestern

Resident Team

From a Raspberry Pi Zero mounted on a Northwestern University baseball cap to a Google Glass application, Memento grew from a practicum course in intelligent information systems to a startup incubating at the Garage Northwestern. Originally intended as a face reminder assistant for easing awkwardness in social and networking situations, Memento has since grown into a system that provides users with augmented cognition and surroundings awareness.
